# Atelier Photobooth 📸✨

An elegant, light, and minimalist web-based photobooth experience built with Next.js and Framer Motion.

## Features

- **Live Camera Feed**: Real-time preview with horizontal mirroring.
- **Interactive Capture**: 3-second countdown for capturing perfect moments.
- **Multiple Layouts**: Choose between "Classic Trio" (3-Vertical), "Modern Quad" (4-Vertical), and "Grid Edition" (2x2).
- **Customizable Aesthetics**: Toggle between curated background colors (White, Off-White, Black, Sage, Blush).
- **Instant Download**: High-resolution photo strip generation using native Canvas API.
- **Responsive Design**: Elegant floating navigation and mobile-optimized studio layout.
- **Animations**: Smooth transitions and interactive carousels powered by Framer Motion.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: Vanilla CSS (Custom Minimalism)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Language**: TypeScript

## Getting Started

Follow these steps to get a local copy up and running.

### Prerequisites

- Node.js (v18.x or later recommended)
- npm (installed with Node.js)

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/atelier-photo.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd atelier-photo
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Run the development server**
   ```bash
   npm run dev
   ```

5. **Open the application**
   Navigate to [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app/`: Next.js App Router routes (Landing, Studio).
- `src/components/`: Reusable UI components (Navbar, etc.).
- `src/app/globals.css`: Custom minimalist design system and responsive utilities.
